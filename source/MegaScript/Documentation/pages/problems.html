Problems

Here is a list of the known problems in this version.

  1. Object callbacks can currently only have one argument due to the way that they are encoded.
  2. Strings are still pushed onto the stack as raw pointers. They simply point to the string in the code.
  3. Library files as well as regular MegaScript files don't load correctly on Windows.
  4. Error catching is not entirely finished. Some errors may freeze the program, or cause a segmentation fault.
