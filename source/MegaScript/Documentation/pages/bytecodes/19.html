FloatOpcode (19)

Pushes a floating point number onto the stack.

The float bytecode is used to create all floating points regardless of size. After the bytecode, one byte is used to define the float size in bytes. Then, the specified amount of bytes is read from the memory, and the program counter is incremented by that same size. Finally, the extracted bytes are casted to a float, and it is then pushed onto the stack.
