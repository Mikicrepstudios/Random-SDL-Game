IntegerOpcode (18)

Pushes an unsigned integer onto the stack.

The integer bytecode is used to create all unsigned integers regardless of size. After the bytecode, one byte is used to define the integer size in bytes. Then, the specified amount of bytes is read from the memory, and the program counter is incremented by that same size. Finally, the extracted bytes are casted to an unsigned integer, and it is then pushed onto the stack.
