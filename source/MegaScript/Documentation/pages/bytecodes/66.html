StandardCallOpcode (66)

Executes a standard call based on a 16 bit ID system.

The standard call bytecode is a useful tool for adding new standard functions to the MegaScript interpreter. When one of these standard calls is parsed by name, it is converted into a 16 bit ID, which saves space. Because this ID is 16 bit (2 bytes), it means that you can have up to 65,536 unique built-in standard calls.
