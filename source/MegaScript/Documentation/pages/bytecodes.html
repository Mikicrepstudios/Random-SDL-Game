Bytecodes

Bytecodes are code that the interpreter can read.

A bytecode is simply a byte which tells a script what to do. Since a bytecode is one single byte, it can only store a value from 0 to 255. (256 possibilities in total) This page will list all of the bytecodes that are currently defined in MegaScript. It will also detail weather a bytecode is used, unused, or currently in development.

Statuses

Status Meaning
Depricated Kept for backwards compatibility.
Dysfunctional Has unfixed bugs/errors.
Internal Used internally by MegaScript; not meant to be changed.
Undefined Not yet created; no functionality.
Unused Not used in the program; likely to be replaced
Unfinished Still in development.
Working Fully functional and supported.
